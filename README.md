# 🎱 Rack It Up

A clean, mobile-friendly **8-Ball Pool Score Tracker** built with vanilla HTML, CSS, and JavaScript. Inspired by Google's Material Design 3.

🌐 **Live Site:** [https://YOUR-USERNAME.github.io/rack-it-up](https://YOUR-USERNAME.github.io/rack-it-up)

---

## Features

- **Match Setup** — Enter player names and choose a race (First to 5, 10, 15, or 21)
- **Match Timer** — HH:MM:SS with Start / Pause / Reset controls
- **Score Tracking** — Large tap-friendly ± buttons for frames won
- **Advanced Stats** — Track 🍎 Apple (Break & Runs) and C+ (Reverse Clearances) per player
- **Win Detection** — Auto-pauses timer and fires a winner modal with confetti
- **Responsive** — Works on desktop and mobile (great next to the pool table)

## Tech Stack

Pure HTML + CSS + JavaScript — no frameworks, no build step, no dependencies (except Google Fonts via CDN).

## Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Save — your site will be live at `https://YOUR-USERNAME.github.io/rack-it-up`

## Local Development

Just open `index.html` in any browser — no server required.
